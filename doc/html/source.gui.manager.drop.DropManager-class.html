Class DropManager

source code

Manages drag and drop within the app. (Doesn't manage drag and drop between apps.) Similar to Carbon Drop Manager.

Source: the control where drag started Target: the control where drag ended (dropped) Controlee: the drawable being dragged.

Enforces: only one drag and drop operation at a time.

For now, each object must implement drop() method. TODO query each object for acceptance of drop.
