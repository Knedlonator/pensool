Slide this layout spec orthogonally.

slide_follow(self, controlee, pixels_off_axis)

source code 

Slide this layout spec orthogonally.

This limits sliding and follows controllee's curve.

Algorithm: we know the cursor moved just a little. Brute force search a pattern in the direction of movement, looking for hits on the controllee.
