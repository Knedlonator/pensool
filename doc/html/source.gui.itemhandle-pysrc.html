112      ''' 
113      Mouse exited item. 
114      This is a filtered event from super GuiControl. 
115      Assert no button down (not a drag.) 
116       
117      Dynamic orthogonality. 
118      A handle menu MIGHT follow the mouse in orthogonal directions, 
119      if the handle menu is a TrackingHandleMenu AND we didn't hit a stop 
120      (the limit of an edge of the controlee morph.) 
121      Exit from item is EITHER in "menu exit" direction 
122      OR in "menu change item" direction. 
123      ''' 
124      # Calculate vector in DCS of mouse exit. 
125      center = self.bounds.center_of() 
126      exit_vector = vector.Point(event.x, event.y) - center 
127      # normalize to manager's (the menu's) axis 
128      norm_vect = vector.normalize_vector_to_vector(exit_vector, self.group_manager.layout_spec.vector) 
129       
130      if norm_vect.y > 10 or norm_vect.y < -10: # FIXME not hardcoded 
131        # exited the side, which means close menu 
132        self.close_manager() 
133      else: 
134        # Exited "change item" side of menu item. 
135        # Tell manager to decide next action (another item, or close menu.) 
136        self.group_manager.do_change_item(event, exit_vector)
