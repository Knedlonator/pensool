zoom(self, direction, event)

source code 

Zoom is op on viewing transformation, i.e. the top transformation. It zooms the model and other controls that might be visible. TODO the bounding box. Note that some morphs contain controls but those controls are transformed by their morph. FIXME and of certain controls such as handles?

This encapsulates knowledge of what scheme objects need to be transformed, and also the knowledge of the step size.
