Module Hierarchy

  • source
    • source.base: Base i.e. foundation modules
      • source.base.alert: Thin wrapper around informative dialogs with user for error conditions. See also excepthook that catches most exceptions and calls them bugs.
      • source.base.bounds: Bounding boxes (rectangles).
      • source.base.command: Command pattern
      • source.base.orthogonal: Geometry: orthogonality.
      • source.base.timer: Timer class. Wraps gobject timer.
      • source.base.transform: Miscellaneous transform and hierarchical modeling.
      • source.base.vector: 2D vector math
    • source.clipboard: Clipboard wrapper. Wraps clipboard of OS/window system.
    • source.composite: Composite drawables. A transforming group (container) of drawables.
    • source.config: Global configuration variables and constants.
    • source.controlinstances: Build control instances for the app.
    • source.decorators: Most Pensool decorators for debug, transform, invalidate. See also picking.
    • source.drawable: Drawable class
    • source.edit: Edit operations: cut, copy, paste. Glue between app and clipboard.
    • source.gui: Graphical User Interface. Custom controls (widgets) alternative to GTK's menus.
      • source.gui.backgroundcontrol: A control in the background. The root control, in charge when no other controls.
      • source.gui.boundingbox: A displayed box bounding the operand for feedback to user.
      • source.gui.control: Base class for custom controls.
      • source.gui.itemcontrol: Control that is an item in a cooperating group (a menu item.)
      • source.gui.itemhandle: Base class for iconic items in a handle menu.
      • source.gui.itemhandlecoords: Items in handle menu. Items that move and resize controlee.
      • source.gui.itemhandleline: Item in a handle menu that creates morphs (draws) when user drags. TODO name should be itemhandledraw.py
      • source.gui.itemmenu: MenuItem class
      • source.gui.manager: Managers coordinate the GUI
        • source.gui.manager.control: Manager of controls to insure only one is active: receiving events, being drawn, and it's controlee focused.
        • source.gui.manager.drop: Coordinates drag and drop, which crosses controls.
        • source.gui.manager.fade: Fade manager: persists handles for a short period after pointer moves away. So user can grab handles that are remote from the edge (for curve tangent lines and center handles.) Fade has only one step, it is not a slow dissolve.
        • source.gui.manager.focus: FocusManager coordinates focus, including:
        • source.gui.manager.handle: Handle manager: ensure only one handle set is active.
        • source.gui.manager.pointer: Pointer manager: state machine for pointer motion The pointer must stop (remain within a few pixels for a short period) before mouseover (picking) is done.
        • source.gui.manager.textselect: Manager of text selection controls.
      • source.gui.menu: Managers of groups of controls, enforces policy: one active item in a menu. There is another manager that enforces a policy over the whole application.
      • source.gui.menuhandle: The Handle subclass of menu, i.e. a menu of type "handle menu". See itemhandle.py for the HandleItem subclass of MenuItem.
      • source.gui.menutraditional: Classic vertical, linear, static, non-tracking menu.
      • source.gui.textselectcontrol: Text selection control (i.e. a selection). A zero-length selection is an insertion bar. Pensool only has selections in text.
    • source.layout: LayoutSpec class.
    • source.morph: Morphs (transforming, composite drawables) and glyphs (primitive drawables.)
      • source.morph.glyph: Drawable primitives, primitive, unit shapes, non-transforming. Note glyphs are not just text characters as in cairo, however glyphs are primitives as in cairo. See morph.py for discussion of strategy for instantiating.
      • source.morph.morph: Morph: composite, transforming drawables (containers) of other morphs or primitive glyphs. Synonyms: shapes, forms, symbols.
      • source.morph.textglyph: Primitive glyph of text (strings). Not a letter in the common meaning for glyph.
      • source.morph.textmorph: Composite drawable of text and its frame.
    • source.pensool: Main of Pensool 2D drawing app.
    • source.pixmap: def draw_background(self, acv, dw, x, y, ww, hh, pb): gc = dw.new_gc() pw = pb.get_width() ph = pb.get_height() offset_x = x % pw offset_y = y % ph dw_y = -offset_y while dw_y < hh: dw_x = -offset_x while dw_x < ww: dw.draw_pixbuf(gc, pb, 0, 0, dw_x, dw_y) dw_x += pw dw_y += ph
    • source.port: Port class
    • source.scheme: Container of things to be drawn (morphs and controls.)
    • source.share: Unaltered, reliable code borrowed off the net.
      • source.share.gui_gtkexcepthook
    • source.style: Style class
    • source.transformer: Transformer: transforms its members.
